# Time Series Analysis Pipeline

## Project Description
This project is an interactive Time Series Analysis application built using Streamlit.

It implements a full pipeline for time series analysis:

Data → Stationarity → Spectrum → ARIMA → Validation → Forecasting

The goal is to analyze and forecast time series data in an interactive and visual way.

---

## Features
- Upload and explore time series data
- Stationarity testing (ADF test)
- Data transformation (Box-Cox)
- Spectral analysis (frequency domain)
- ARIMA modeling for forecasting
- Model validation
- Interactive graphs (Plotly + Matplotlib)

---

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- SciPy
- Statsmodels
- Matplotlib
- Plotly

---

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run TS_app.py
